📖 Guide — Comment mettre en ligne une nouvelle création ?
  1. Remplis le formulaire ci-dessous (photo, nom, description, prix, page)
  2. Clique Générer le bloc
  3. Clique Copier pour copier le code HTML
  4. Va sur github.com → ton dépôt anna-creations
  5. Ouvre le fichier de la bonne page (ex : boucles.html)
  6. Clique l'icône ✏️ (Edit), trouve la ligne <!-- FIN CATALOGUE -->
  7. Colle ton bloc juste avant cette ligne
  8. Clique Commit changes → c'est en ligne !
